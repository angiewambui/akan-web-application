AKAN NAME APP

AKAN NAME WEB APPLICATION

This is a webpage that allows users to enter their birthdates,their gender and once click me button is clicked,the Akan name is displayed on a page. Akan names are derived from Ghanian culture. Frequently in Ghana, children are given their first name as a 'day name which corresponds to the day in the week they were born. Here are Ghanian day names for both male and female:

males names

  • Sunday: Kwasi
  • Monday: Kwadwo
  • Tuesday: Kwabena
  • Wednesday: Kwaku
  • Thursday: Yaw
  • Friday: Kofi
  • Saturday: Kwame
>

female names

  • Sunday; Akosua
  • Monday: Adwoa
  • Tuesday: Abenaa
  • Wednesday: Akua
  • Thursday: Yaa
  • Friday: Afua
  • Saturday: Ama
